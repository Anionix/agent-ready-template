# Changelog

All notable changes to this repository are recorded here.

This project uses Semantic Versioning for tagged releases.

## v0.1.1 - 2026-05-17

- Added Sigstore keyless source release signing for release artifacts starting
  with `v0.1.1`.
- Added OpenSSF Best Practices Silver evidence for private continuity,
  release-artifact signing, and honest solo-maintainer limits.

## v0.1.0 - 2026-05-17

- Established the public agent-ready repository template contract.
- Added CI, CodeQL advanced setup, OpenSSF Scorecard, secret scanning, workflow pin auditing, Dockerfile pin auditing, Hadolint, ShellCheck, and optional fuzzing evidence.
- Added optional sandbox and runtime profile documentation without making those runtimes default dependencies.
- Added OpenSSF Best Practices evidence documentation, a dynamic status badge, and public process files.

## Earlier work

- Initial template, agent instructions, adapter notes, readiness checks, and public-release guardrails were developed before the first tagged release.
