# CONVENTIONS.md

## English

This repository uses `AGENTS.md` as the source of truth for AI coding agents.

Aider users should read this file and `AGENTS.md` before making changes.

## 日本語

このリポジトリでは `AGENTS.md` をAI coding agent向けの正本として扱います。

Aiderを使う場合は、変更前にこのファイルと `AGENTS.md` を読んでください。

## Coding conventions / コーディング規約

- Prefer small, focused changes.
- Match existing style.
- Add or update tests when behavior changes.
- Run `mise run check` before finishing.

- 小さく焦点の定まった変更を優先する。
- 既存のstyleに合わせる。
- 挙動が変わる場合はtestを追加または更新する。
- 完了前に `mise run check` を実行する。
