# Roadmap / ロードマップ

## English

This clean template does not import the historical issue backlog, project
boards, Rust migration work, AI evaluation lane, or v0.2 hardening queue from
`Anionix/agent-ready-repo-template-spec`.

Initial follow-up for `Anionix/agent-ready-template` is intentionally small:

- Revalidate the OpenSSF Best Practices project for this new repository.
- Add a live badge only after the new project reports Silver.
- Keep `main` protected and CI green.
- Publish future template changes as small, reviewable SemVer releases.

New roadmap items should be created in this repository only when they are part
of the canonical template contract. Research, experiments, and v0.2+ hardening
backlog stay in `agent-ready-repo-template-spec`.

## 日本語

このclean templateには、`Anionix/agent-ready-repo-template-spec` の過去issue
backlog、project board、Rust migration、AI evaluation lane、v0.2 hardening
queueは取り込みません。

`Anionix/agent-ready-template` の初期follow-upは小さく保ちます。

- 新repo向けに OpenSSF Best Practices project を再検証する。
- 新projectがSilverを示してからlive badgeを追加する。
- `main` protection と CI green を維持する。
- 今後のtemplate変更は、小さくreview可能なSemVer releaseとして出す。

canonical template contractに入れるものだけを、このrepositoryのroadmap itemに
します。research、experiment、v0.2+ hardening backlogは
`agent-ready-repo-template-spec` 側に残します。
