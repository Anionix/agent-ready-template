# OpenSSF Best Practices Proposal Notes

This clean template was seeded from the `agent-ready-repo-template-spec`
`v0.1.1` Silver baseline, but it does not carry forward the source repository's
editable Best Practices proposal URLs.

Before adding a live badge or submitting evidence for this repository:

1. Create or update the OpenSSF Best Practices project for
   `Anionix/agent-ready-template`.
2. Regenerate proposal links against that project and this repository URL.
3. Confirm the Best Practices page reports Silver for this repository.
4. Only then add a live badge to `README.md`.
