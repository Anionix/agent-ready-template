# Adoption checklist / 導入チェックリスト

## English

- [ ] Create a project repository with “Use this template”.
- [ ] Customize `AGENTS.md`.
- [ ] Customize `mise.toml`.
- [ ] Customize `compose.yaml`.
- [ ] Run `mise install`.
- [ ] Run `mise run doctor`.
- [ ] Run `mise run agent-ready`.
- [ ] Run `mise run release-check` before public release.
- [ ] Enable GitHub template repository setting.
- [ ] Decide whether optional Dev Container is needed.
- [ ] Decide whether sandbox is needed for high-risk agent work.

## 日本語

- [ ] “Use this template” でproject repoを作る。
- [ ] `AGENTS.md` を調整する。
- [ ] `mise.toml` を調整する。
- [ ] `compose.yaml` を調整する。
- [ ] `mise install` を実行する。
- [ ] `mise run doctor` を実行する。
- [ ] `mise run agent-ready` を実行する。
- [ ] 公開前に `mise run release-check` を実行する。
- [ ] GitHubのtemplate repository設定を有効化する。
- [ ] optional Dev Containerが必要か決める。
- [ ] 高リスクagent作業にsandboxが必要か決める。
